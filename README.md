# astrbot_eryou_daily

AstrBot 插件：用 `/委托` 检查崩坏：星穹铁道今日每日实训和派遣状态。

## 功能

- `/委托`：检查绑定账号的今日状态。
- `/委托绑定 <米游社Cookie>`：绑定当前 QQ/平台用户到一个星铁账号。
- `/委托解绑`：删除当前用户的本地绑定。
- `/委托帮助`：显示用法。

第一版只支持崩坏：星穹铁道国服。插件会先检测当前发送者是否绑定账号；没绑定时不会请求米游社接口。

## 安装

1. 将本仓库作为 AstrBot 插件安装，或下载后放入 AstrBot 插件目录。
2. 在 AstrBot WebUI 中重载插件。
3. 在 QQ 私聊里发送 `/委托帮助` 测试插件是否响应。

## 绑定

建议私聊机器人发送绑定指令，不要在群里暴露 Cookie。

```text
/委托绑定 cookie_token=...; account_id=...
```

绑定成功后发送：

```text
/委托
```

插件会返回：

```text
星铁今日委托检查
账号：角色名（UID 100000000）
每日实训：500/500，已完成
派遣：4/4，已全部派出
今天的每日已经 Clear。
```

## 本地数据

绑定信息保存在插件目录的 `data/bindings.json`，该文件不会提交到 Git。

## 排错

- 提示未绑定：先私聊发送 `/委托绑定 <Cookie>`。
- 提示 Cookie 失效：重新从米游社获取 Cookie 后再绑定。
- 群里没有反应：确认插件已启用，并且消息是以 `/委托` 或 `／委托` 开头。
- 米游社接口可能调整校验参数；如果突然全部查询失败，需要更新插件。
