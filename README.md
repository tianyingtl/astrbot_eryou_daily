# astrbot_eryou_daily

AstrBot 二游每日检查插件。当前版本先支持崩坏：星穹铁道，用 `/委托` 检查今日每日实训状态。

## 功能

- 在 QQ 群或私聊发送 `/委托`，检查星铁今日每日实训是否 Clear。
- 支持米游社账号绑定。绑定一次米游社账号，后续接入其它米家游戏时可以复用，不需要重复登录。
- 群聊里可以发起绑定；二维码和登录说明会由机器人私聊发送。
- 支持扫码登录入口；手机号登录入口会提示说明，当前公开版不直接接收验证码。
- 支持 WebUI 配置群白名单/黑名单。
- 支持 `/委托设置 星铁 20:00`，到点未完成时在当前群 at 提醒。

## 安装

1. 将本仓库放入 AstrBot 插件目录，或通过 AstrBot 插件管理安装。
2. 安装依赖：

```bash
pip install -r requirements.txt
```

3. 在 AstrBot WebUI 中重载插件。
4. 私聊机器人发送 `/委托帮助`，确认插件有响应。

## 用户使用

### 检查每日实训

```text
/委托
```

未绑定时，机器人会提示先私聊绑定。

### 绑定账号

群聊里或私聊发送：

```text
/委托绑定
```

如果是在群聊触发，机器人会把后续绑定说明私聊发给你，群里只显示回执。

按这个顺序操作：

```text
/委托绑定
/委托绑定 星铁
/委托扫码
```

如果这些命令是在群里发送，机器人会把二维码私聊发给你。用米游社 App 扫码并确认登录后，再发送：

```text
/委托确认
```

绑定成功后，就可以在群聊或私聊发送 `/委托` 查询状态。

### 设置提醒

提醒必须在群里设置，因为插件需要知道要在哪个群 at 你。

```text
/委托设置 星铁 20:00
```

格式：

```text
/委托设置 游戏名 时间
```

时间使用 24 小时制 `HH:MM`，例如 `08:30`、`20:00`。如果到点时星铁每日实训还没满，机器人会在设置提醒的群里 at 你。

当前支持的游戏名：`星铁`。

### 手机号登录

```text
/委托手机号
```

当前公开版不会直接收手机号和短信验证码。米游社手机号登录通常涉及人机验证，公开机器人直接处理验证码不安全也不稳定。推荐使用 `/委托扫码`。

### 解绑

```text
/委托解绑
```

会删除当前 QQ/平台用户在本插件里的本地米游社账号和游戏绑定。

## 当前支持的游戏

- 崩坏：星穹铁道：`/委托绑定 星铁`

其它米家游戏后续可以接入同一个米游社账号体系。

## 数据保存

绑定信息保存在插件目录：

```text
data/bindings.json
```

该目录已被 `.gitignore` 忽略。公开仓库时不要提交真实 Cookie、绑定文件、登录二维码截图或包含账号凭证的日志。

## WebUI 设置

插件提供 `_conf_schema.json`，可以在 AstrBot 插件设置页配置：

- `group_filter_mode`：`关闭` / `白名单` / `黑名单`
- `whitelist_groups`：白名单群号列表
- `blacklist_groups`：黑名单群号列表

白名单模式下，只有白名单群能使用 `/委托` 相关命令。黑名单模式下，黑名单群不能使用。

## 常见问题

- 群里发送 `/委托绑定`：机器人会把游戏选择说明私聊发给你。
- 群里发送 `/委托扫码`：机器人会把米游社二维码私聊发给你。
- `/委托绑定 星铁`：机器人会让你选择扫码或手机号登录。
- `/委托确认`：扫码确认后完成绑定。
- `/委托设置 星铁 20:00`：设置每日实训提醒。
- 如果机器人没法主动私聊你，请先私聊机器人发送 `/委托绑定`。
- `/委托` 提示未绑定：先私聊完成绑定流程。
