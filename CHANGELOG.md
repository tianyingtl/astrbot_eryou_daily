# 更新日志

## v1.0.7

- `/委托绑定 异环` 不再读取可能失效的旧塔吉多登录态，改为直接私聊发送重新登录教程。
- `/委托绑定 异环 UID` 会保留指定 UID，并在短信登录成功后绑定对应角色。
- 异环查询遇到登录态失效时，会明确提示重新绑定，不再只显示 HTTP 402。

## v1.0.6

- 跟随塔吉多官网当前客户端，将接口协议版本更新为 `1.2.5`。
- 塔吉多接口返回 HTTP `401`、`402` 或 `403` 时，自动刷新登录态并重试原请求一次。
- 刷新后的 access token 和 refresh token 会继续写回绑定数据，避免下一次查询重复失败。

## v1.0.5

- 绑定数据迁移到用户目录：`%USERPROFILE%\.astrbot_eryou_daily\bindings.json`。
- 旧版插件目录内的 `data/bindings.json` 存在时，会自动复制到新位置。
- 异环“今日活跃”改名为“活跃度”，和游戏内名称保持一致。
- 异环支持 `/委托绑定 异环 UID` 指定角色，多角色账号不再默认选择第一个。
- 异环提醒改为检查活跃度；周日额外检查都市活力是否清完。
- 查询和提醒文案改为“娜娜米提醒”风格，移除不清楚的英文完成提示。

## v1.0.4

- 新增体力/资源显示：星铁开拓力、原神原粹树脂、绝区零电量。
- 新增异环绑定和查询：塔吉多手机号短信登录、活跃度、本性像素、都市活力。
- 异环支持 `/委托 异环` 和 `/委托设置 异环 HH:MM`。

## v1.0.3

- 删除手机号/验证码登录入口中的米游社验证码方案。
- `/委托绑定 游戏名` 改为直接发送米游社扫码二维码。
- README 增加插件市场展示图和群内提醒效果图。
